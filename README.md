# Table ronde - Index des fichiers

Ce repository contient les supports d'animation pour une table ronde autour de la methode BMAD.

## Fichiers
- `Fiche_Animateur_BMAD.md` : guide complet de l'animateur (structure, script minute par minute, conclusion).
- `pense-bete.md` : version courte et operationnelle pour animer en direct.
- `post-its.md` : contenu/preparation des idees a afficher et manipuler pendant l'activite.

## Utilisation conseillee
1. Lire `Fiche_Animateur_BMAD.md` pour preparer la session.
2. Garder `pense-bete.md` ouvert pendant l'animation.
3. Utiliser `post-its.md` pour preparer et organiser les contributions.
